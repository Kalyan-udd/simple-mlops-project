How the Model Works

1

Image Processing

Genuine and forged banknote specimens are digitized. A Wavelet Transform tool extracts core numerical features from the images.

2

Model Training

A Gradient Boosting Classifier is trained on a dataset of thousands of notes to learn the hidden mathematical patterns of forgery.

3

Live Inference

Using our FastAPI backend, you can pass new data points into the model and receive a high-accuracy prediction in milliseconds.
